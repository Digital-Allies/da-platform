# Icon Component

Display custom Atomic Finds PNG icons from the assets library.

## Usage

```jsx
<Icon name="leaf" size={48} />
<Icon name="star" size={32} />
<Icon name="heart" size={24} />
```

## Available Icons

**Product & Furniture:**
- bamboo-chair, bamboo-lamp, woven-basket, rattan-shelf, plant-stand, mirror, stool

**Navigation & UI:**
- home, shop, search, cart, user, heart, check

**Celestial & Brand:**
- star, crescent, sun, planet, leaf

## Props

- `name` — Icon filename without .png extension (default: "leaf")
- `size` — Icon size in pixels (default: 48)
- `className` — Optional CSS class
- `style` — Optional inline styles
