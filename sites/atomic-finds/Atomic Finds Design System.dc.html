Usage Guidelines

Color Rule

95% warm neutrals (rattan, charcoal, cream). 5% golden accent glow (#ffd966, #d4822a). Maintains visual calm and readability.

Icon Style

Custom SVG icons. Consistent stroke weight. Botanical and celestial themes. 56px minimum size for UI, scale as needed for hero imagery.

Pattern Use

Overlay patterns as background textures or card accents. Layer at low opacity (8-15%) to maintain legibility. Bamboo and rattan patterns primary.

Spacing

20px baseline grid unit. Use multiples (20, 40, 60, 80px) for section padding. 16px for internal card spacing. Consistent rhythm.

Borders

Use #d4aa82 (rattan) for primary borders. 1-2px weight. Subtle, functional. Radius 0-8px. Corners stay mostly square.

Dark Mode

Primary aesthetic. All surfaces dark (#0a0a0a to #1a1510). Warm text (#f0e6d6). Golden glows on headlines. No light mode variant.
