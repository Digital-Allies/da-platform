THE WORKFLOW

Build here → ship to Vercel

There is no live sync between Figma Make and this workspace. Importing a file either direction is a one-time static snapshot, not a connection — so keep one source of truth instead of two drifting apart. Since the goal is prototypes that go to Vercel, that source of truth is here.

COMPONENTS · THE HERO

Galaxy Card — the signature

A glowing orbital ring wraps one continuous rounded card. The moon rides a tilted orbit — sweeping in front along the bottom, slipping behind across the top. Image, title, script tagline, description, and price flow inside. Stars twinkle in the image well.

These are live components from the design system bundle. Use them in your prototypes by importing the AtomicFindsDesignSystem_af0620 namespace.
