04Composition rules

  • Section padding is generous — typically 80px top and bottom.
  • Max width 1024px for prose, 1280px for grids and footers.
  • One accent per view. Break symmetry only with the red dot or a red underline — never with blue or pink.
  • Everything sits on the 20 px Technical Lace grid. See spacing.
