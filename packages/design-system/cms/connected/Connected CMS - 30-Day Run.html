How to run this without burning out

  • One card a day. Each day is a single focus. Do it, tick it, close the laptop.
  • Stop at 135. If the timer's up mid-task, leave a note in your commit and pick it up tomorrow. The plan has slack.
  • Every build day ends two ways: a commit pushed to GitHub, and a green deploy on Vercel. If both are true, the day shipped.
  • "Done when —" lines are your acceptance check. Don't tick the day complete until its Done-when is true.
  • Weekends are review, not building. Skim, confirm, breathe. The work waits.
  • You're driving Claude. Items tagged Code mean "have Claude Code write it" — your job is to point, paste, and check the result, not to type the code.
