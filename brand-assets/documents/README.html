Overview

What This Repo Is

Source for digitalallies.net — a single-file HTML website built without frameworks, dependencies, or build tools. Fast, accessible, bilingual, and structured for both traditional search engines and AI assistants.

This repo is a reference document for the Digital Allies brand system and a deployable static site. Nothing requires compilation. Open any .html file directly in a browser and it works.
Stack

Technical Foundation

Markup HTML5 Semantic elements, ARIA roles, landmark regions, proper heading hierarchy throughout.
Styles Tailwind CSS (CDN) No build step. Extended config via inline script block with DA color tokens and font stacks.
Logic Vanilla JS LanguageController, schema generators, interactive checklists, video filters. No React, no Vue.
Icons Lucide (CDN) Inline SVG icon set. Re-initialized after language toggles to preserve icon nodes.
Fonts Google Fonts Lexend Deca (headers) + JetBrains Mono (body/details). Preconnect tags on all pages.
Forms FormSubmit No backend. Honeypot field, auto-response, and redirect configured via hidden inputs.

Dev

Running Locally & Deploying

No build step. No npm install. Tailwind and Lucide load from CDN.

# Option 1 — Node static server npx serve . # Option 2 — Python python3 -m http.server 8080 # Then open http://localhost:8080

After deploying, submit updated pages for reindexing via Google Search Console:

https://digitalallies.net/ https://digitalallies.net/learn/seo-aeo https://digitalallies.net/learn/alttext

Validate JSON-LD at validator.schema.org after any schema changes.
