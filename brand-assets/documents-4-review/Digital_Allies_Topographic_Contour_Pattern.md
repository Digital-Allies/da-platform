# Digital Allies — Topographic Contour Pattern System

8 transparent PNG graphic assets. All use **Charcoal #2D2D2D** contour lines on a fully transparent background. Accent dots are **Pulse Blue #3A7BD5** or **Signal Red #C5301A** as specified.

---

## Variation Usage Guide


| #   | Name                      | Ratio    | Accent         | Best Used For                                  |
| --- | ------------------------- | -------- | -------------- | ---------------------------------------------- |
| 1   | **Full Field — Dense**    | 1:1      | Pulse Blue dot | Section backgrounds, hero panels, social tiles |
| 2   | **Full Field — Sparse**   | 1:1      | Signal Red dot | Feature cards, open-layout web sections        |
| 3   | **Horizon Strip — Wide**  | 3:2 wide | Pulse Blue dot | Page dividers, email banners, LinkedIn headers |
| 4   | **Vertical Edge Band**    | 2:3 tall | None           | Page margin accents, case study sidebars       |
| 5   | **Corner Cluster**        | 1:1      | None           | Card corners, pull quote decoration            |
| 6   | **Text-Wrap Halo**        | 1:1      | None           | Overlay behind body copy, stat callouts        |
| 7   | **Peak Marker**           | 1:1      | Signal Red dot | Section headers, icon spots, bullet markers    |
| 8   | **Faded Field — Texture** | 1:1      | None           | Full-page background overlays, hero texture    |


---

## Brand Notes

- **The single dot = the peak.** It's the "X marks the spot" — the SEO goal, the answer found.
- Pulse Blue dots: forward motion, digital elevation, primary CTA contexts
- Signal Red dots: urgency, achievement, standout moments
- Layer the **Faded Field** (#8) behind text at 30–50% opacity for depth without distraction
- The **Corner Cluster** (#5) pairs well with the **Horizon Strip** (#3) on a single page layout
