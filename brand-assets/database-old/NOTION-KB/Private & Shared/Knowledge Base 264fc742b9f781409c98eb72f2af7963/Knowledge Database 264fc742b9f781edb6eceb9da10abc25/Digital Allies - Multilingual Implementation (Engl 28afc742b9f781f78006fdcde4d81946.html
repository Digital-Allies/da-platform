Language Switching System

Implementation Architecture

  • Attribute-based: data-en and data-es attributes on elements
  • Client-side switching: JavaScript toggles between languages
  • State persistence: Language preference stored in localStorage
  • Fallback support: Graceful degradation if JavaScript disabled

JavaScript Language Controller

Core Functionality

class LanguageController {
  constructor() {
    this.currentLang = localStorage.getItem('language') || 'en';
    this.init();
  }
  
  init() {
    this.updateContent(this.currentLang);
    this.updateUI(this.currentLang);
    this.updatePageLang(this.currentLang);
  }
  
  toggleLanguage(lang) {
    this.currentLang = lang;
    localStorage.setItem('language', lang);
    this.updateContent(lang);
    this.updateUI(lang);
    this.updatePageLang(lang);
  }
  
  updateContent(lang) {
    const elements = document.querySelectorAll('[data-en][data-es]');
    elements.forEach(el => {
      const content = el.getAttribute(`data-${lang}`);
      if (content) {
        el.textContent = content;
      }
    });
  }
  
  updateUI(lang) {
    // Update language toggle button states
    document.querySelectorAll('[id^="lang-"]').forEach(btn => {
      btn.classList.remove('text-primary-blue', 'bg-white');
      btn.classList.add('text-gray-dark');
    });
    
    const activeBtn = document.getElementById(`lang-${lang}`);
    if (activeBtn) {
      activeBtn.classList.add('text-primary-blue', 'bg-white');
      activeBtn.classList.remove('text-gray-dark');
    }
  }
  
  updatePageLang(lang) {
    document.documentElement.lang = lang;
  }
}

Cultural Considerations

Content Adaptation

  • Currency: USD pricing maintained (local market)
  • Contact information: Phone format localized for US/Mexico
  • Business hours: Mountain Standard Time clearly indicated
  • Legal pages: Separate Spanish versions for compliance

Language-Specific Links

function updateLegalLinks(lang) {
  const legalLinks = {
    'privacy': lang === 'es' ? '/privacy-es.html' : '/privacy.html',
    'terms': lang === 'es' ? '/terms-es.html' : '/terms.html'
  };
  
  Object.entries(legalLinks).forEach(([type, url]) => {
    const links = document.querySelectorAll(`[data-legal="${type}"]`);
    links.forEach(link => link.href = url);
  });
}

Language Announcement

function announceLanguageChange(lang) {
  const announcement = lang === 'es' 
    ? 'Idioma cambiado a español'
    : 'Language changed to English';
    
  const announcer = document.createElement('div');
  announcer.setAttribute('aria-live', 'polite');
  announcer.setAttribute('aria-atomic', 'true');
  announcer.className = 'sr-only';
  announcer.textContent = announcement;
  
  document.body.appendChild(announcer);
  setTimeout(() => document.body.removeChild(announcer), 1000);
}

Performance Optimization

Efficient Loading

  • Single page load: No page refresh required for language switch
  • Cached content: Both languages loaded simultaneously
  • Minimal JavaScript: Lightweight language switching logic
  • Progressive enhancement: Works without JavaScript (default English)

Memory Considerations

  • Attribute storage: Efficient DOM attribute usage
  • State management: Minimal localStorage usage
  • Event delegation: Single event listener for language toggles
