What the CMS does

Three core jobs
01 Website page editing Build and edit pages by stacking sections from a component library — hero, departments, field notes, rich text, CTA, and more. Reorder, edit inline, draft/publish.
02 Global design editing Edit the brand tokens the whole site reads — colors, type, spacing — from one place. Change once, update every page.
03 Blog & articles The Press Office: write, schedule, and publish blog posts, press releases, and case studies that appear on the site.
