# Archive — superseded CMS prototypes

These are earlier / alternate CMS explorations. **They are not the CMS.**
The single, canonical CMS is **`cms/dashboard.html`** (side-navigation
Connected CMS Dashboard, served by `cms/app.js` + `cms/style.css`).

Everything in this folder is kept for reference only — nothing in the live
project links here. Don't point developers or collaborators at these.

## What's here

- `connected/` — the Next.js/Supabase "Connected CMS" prototype family
  (Connected CMS, Digital Allies CMS workspace, the 30-Day Run app, the
  build plan, the handoff sequence, and the `.ics`). A different, more
  speculative direction than the chosen dashboard.
- `CMS_DASHBOARD_UI.html` — an earlier standalone dashboard mockup.
- `content-calendar.html` — a bundled copy of the 30-Day Run. The live,
  reusable calendar now lives at `ui_kits/website/content-calendar.html`
  and inside the dashboard's Content Calendar tab.
- `CMS_IMPLEMENTATION_PLAN.html` — duplicate of the spec. The canonical
  spec is `CMS_IMPLEMENTATION_PLAN.html` at the project root.

To restore any of these, move them back out of `_archive/`.
